## 2024-05-22 - Performance Anti-Pattern: Permanent Will-Change\n**Learning:** Applying `will-change-transform` permanently via class lists on lists of items (like blog posts) consumes excessive GPU memory. It should only be applied immediately before animation (e.g., hover/focus).\n**Action:** Use CSS pseudo-classes (e.g., `group-hover:will-change-transform`) or JS event listeners to toggle the property dynamically.
