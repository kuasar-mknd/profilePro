**Gamines** est un court-métrage réalisé dans le cadre du **48h Film Project Genève**.

> **Pitch** : Deux sœurs s'isolent dans une forêt d'automne. Mais ce qui devait être un moment de deuil partagé se transforme en tribunal.

## Mon Rôle : Gaffer

J'ai occupé le poste de **Gaffer** (Chef Électricien) sur ce tournage, travaillant en étroite collaboration avec le Chef Opérateur François Verreyt pour sculpter la lumière de cette forêt d'automne.
